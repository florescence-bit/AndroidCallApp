# mycallapp

A simple and elegant calling application for Android.

## Features

*   **Recents:** View your complete call history, including incoming, outgoing, and missed calls.
*   **Contacts:** Browse and search through all the contacts on your device.
*   **Direct Calling:** Make calls directly from the app by tapping on a call log entry or a contact.
*   **Personalized Home Screen:** A clean and modern home screen that greets you and provides quick access to recent calls.
*   **Customizable Settings:**
    *   **Theme Selection:** Choose between System Default, Light, and Dark themes.
    *   **Notification Controls:** Easily enable or disable app notifications.
    *   **Privacy Policy:** A direct link to the app's privacy policy.

## Permissions Used

*   `android.permission.READ_CALL_LOG`: To display the call history.
*   `android.permission.CALL_PHONE`: To initiate phone calls from the app.
*   `android.permission.READ_CONTACTS`: To display the device's contact list.

## Setup

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Build and run the app on an Android device or emulator.
